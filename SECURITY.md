This repository contains **Markdown only** (no executable code in-tree).

## Reporting

- **Issues with this org’s documentation or metadata**: open a **[private security advisory](https://github.com/qbitOS/ugrad/security/advisories/new)** if sensitive.
- **Product vulnerabilities** (web apps, WASM, terminals): report to **[qbitOS/uvspeed](https://github.com/qbitOS/uvspeed)** using its security policy.

## Compliance reference

See [COMPLIANCE.qmd](COMPLIANCE.qmd) and [qbitOS/qbitos-freya](https://github.com/qbitOS/qbitos-freya) for control-envelope context.

## Scope

Do not expect this umbrella repo to track CVEs for the full uvspeed stack—monitor upstream.
