# Syncing this umbrella repo

This repository is **documentation-only**. Canonical **code** lives in [qbitOS/uvspeed](https://github.com/qbitOS/uvspeed).

## Typical workflow

1. Edit **README.md**, **COMPLIANCE.qmd**, **COMPLIANCE.md**, **NOTICES.md** here for org narrative, diligence, and repo tables.
2. Mirror the same changes to **`~/dev/projects/ugrad`** if you use that as a second working copy, or treat **this path** (`/Volumes/qbitOS/00.dev/ugrad`) as the volume source of truth.
3. Commit and push to **GitHub** — see [GITHUB.md](GITHUB.md).

## When upstream docs move

If paths under `qbitOS/uvspeed` change (e.g. `docs/ugrad-*.md`), update **README** “Related documentation” links and **NOTICES.md** here.
