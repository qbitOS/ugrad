# uvspeed μgrad snapshot

Point-in-time copies of **μgrad / ugrad–related paths** from a local [qbitOS/uvspeed](https://github.com/qbitOS/uvspeed) checkout, preserving the same relative layout as the monorepo root (`qbit/`, `web/`, `docs/`, root captures, etc.).

This tree is for **reference, search, and offline review** in the org repo. It is **not** a substitute for a full uvspeed clone: internal imports, tooling, and sibling assets outside this list may still be required to run or build.

To refresh after updating uvspeed, re-run the copy from your checkout (see repo `WORKSPACE.md`).
