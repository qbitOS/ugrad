Terminal install & ops

Companion to games hub · zsh opens files · commands execute in browser

1 · Why ugrad fails in Terminal.app

ugrad games is a HexTerm command (in terminal.html), not a macOS binary. Use the steps below for everyone else: npm bridge, hosted Pages, or clone + npx.

2 · npm ugrad-cli (zsh → open PWA)

From a uvspeed checkout (package updates track repo npm/ugrad-cli):

cd /path/to/uvspeed
npm install -g ./npm/ugrad-cli          # optional global
npx ugrad-cli help
npx ugrad-cli open      # HexTerm — then type: ugrad games | ugrad go | goboard watch
npx ugrad-cli raw       # raw-games-ugrad.html
npx ugrad-cli hub
npx ugrad-cli alphabet

Published (no clone): GitHub Pages HexTerm
